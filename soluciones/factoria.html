The Stack: 4 Microservicios eIDAS

Una API unificada. Cuatro microservicios. Cada uno integra un servicio de confianza eIDAS cualificado, expuesto como primitivas REST.

01

Evidence Manager

Inmutabilidad como servicio

Hashing SHA-256 automático + Qualified Timestamping (TSA). Perfiles XAdES-T/LTA con re-sellado periódico. Verificabilidad offline. Cada archivo, log o dato queda sellado con fecha cierta cualificada.

Presunción de fecha exacta (Art. 41.2 eIDAS)

Inversión de carga de prueba (Art. 326.4 LEC)

Logs Fotos siniestro IoT data APPCC Decisiones IA
02

Signature Manager

Firma como servicio

Niveles AES y QES. Perfiles PAdES, XAdES y CAdES. QSCD cloud (Cloud HSM). SDK mobile. Desde firma avanzada hasta cualificada con certificado. Equivalencia a firma manuscrita en toda la UE.

Equivalencia firma manuscrita (Art. 25.2 eIDAS)

Contratos Consentimientos Albaranes KYC
03

Notification Manager

Entrega electrónica certificada (ERDS)

Servicio de entrega electrónica certificada (ERDS). Prueba de envío, entrega y contenido con sellos cualificados. Idóneo para comunicaciones laborales, regulatorias y contractuales.

ERDS eIDAS (Art. 43/44 Reglamento 910/2014)

Despidos Regulatorias Vencimientos Reclamaciones
04

Custody Manager

Custodia como servicio

Almacenamiento certificado con garantía de integridad a largo plazo. Custodia con garantía de integridad a largo plazo (LTV). Perfiles PAdES-LTV/XAdES-LTA, re-sellado periódico. Nube soberana europea. Preparado para archivo electrónico cualificado cuando entre en vigor.

Integridad a largo plazo (LTV) · eIDAS
