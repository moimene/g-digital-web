The Stack: 4 Microservicios eIDAS

Una API unificada. Cuatro microservicios. Cada uno integra un servicio de confianza eIDAS cualificado, expuesto como primitivas REST.

01

Evidence Manager

Inmutabilidad como servicio

Hashing SHA-256 automatico + Qualified Timestamping (TSA). Cada archivo, log o dato queda sellado con fecha cierta cualificada.

Presuncion de fecha exacta (Art. 41.2 eIDAS)

Inversion de carga de prueba (Art. 326.4 LEC)

Logs Fotos siniestro IoT data APPCC Decisiones IA
02

Signature Manager

Firma como servicio

Firmas XAdES, CMS. Cloud HSM. SDK mobile. Desde firma simple hasta cualificada con certificado. Equivalencia a firma manuscrita en toda la UE.

Equivalencia firma manuscrita (Art. 25.2 eIDAS)

Contratos Consentimientos Albaranes KYC
03

Notification Manager

Notificacion como servicio

Notificaciones por email y SMS con prueba de envio, entrega y contenido. Acuses de recibo con sello de tiempo y firma cualificada.

Entrega Electronica Certificada eIDAS

Despidos Regulatorias Vencimientos Reclamaciones
04

Custody Manager

Custodia como servicio

Almacenamiento certificado con garantia de integridad a largo plazo. Custodia Digital Cualificada (QEA) conforme a eIDAS 2.0. Nube soberana europea.

Custodia Cualificada eIDAS 2.0

Historiales Datasets IA Expedientes Batch records
